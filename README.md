# Restaurant Ordering App

A restaurant ordering application built with HTML, CSS, and JavaScript. Users can browse a restaurant menu, add and remove items from their order, view a dynamically updated total, and complete checkout through a payment modal with form validation.

## Screenshot

![Restaurant Ordering App](images/homepage-screenshot.png)

## Live Demo

https://shwarzbergzelda.github.io/Restaurant-Ordering-App/

## Features

- Dynamic menu rendering from JavaScript data
- Add menu items to an order
- Remove items from an order
- Automatic total price calculation
- Checkout modal with payment form
- Required form validation
- Personalized order confirmation message
- Responsive and centered layout
- Smooth scrolling to order section

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/shwarzbergzelda/Restaurant-Ordering-App.git
```

2. Navigate to the project directory:

```bash
cd Restaurant-Ordering-App
```

3. Open `index.html` in your browser or run the project using a local development server such as Live Server.
